    def self.get(id_or_name) : self #

    returns the first container matching the given ID or name, or raise a Docker::APIClient::NotFound it's not found. In the case that multiple are found, a Docker::APIClient::Exception will be raised. To get multiple matching containers, use Docker::Containers[] or Docker::Containers.containers, which return instead a (possibly empty) list of all matching containers.
