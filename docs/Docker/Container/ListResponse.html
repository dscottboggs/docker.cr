    class Docker::Container::ListResponse

    • Docker::Container::ListResponse
    • Reference
    • Object

    Overview

    an array of ListResponse is received from the docker client when requesting a list of containers. See https://docs.docker.com/engine/api/v1.30/#operation/ContainerList
